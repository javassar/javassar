### Core Research & Projects

#### [Are Large Language Models Creative? (Senior Thesis)](https://github.com/javassar/thesis)
An interdisciplinary cognitive science thesis investigating the conceptual and empirical boundaries of machine creativity.
* **The System:** Engineered an autonomous 7-stage prompt architecture using Claude, ChatGPT, and Gemini to generate 18 fully playable 2D video games from scratch using the Phaser framework.
* **The Evaluation:** Designed and executed a mixed-methods human-in-the-loop behavioral study (N = 81) utilizing a modified Consensual Assessment Technique (CAT) to map how humans attribute creativity to AI-generated outputs.

#### [URSI-LLM-Convo-Bot](https://github.com/jodeleeuw/URSI-LLM-Convo-Bot)
An project initalized collaboratively during an Undergraduate Research Summer Institute (URSI) Fellowship and carried on by me that uses AI to mitigate American reluctance to discuss controversial topics.
* **The System:** Co-developed a specialized conversational LLM system and designed an online jsPsych experiment to iteratively pilot the system.
* **The Data:** Designed and tested a novel survey instrument using large-scale online samples (N > 550), conducting predictive modeling, NLP, and factor/cluster analyses.

#### Riverkeeper Statistical Consulting
A collaborative statistics capstone project empowering a local Hudson River nonprofit with actionable environmental metrics.
* **The Analysis:** Performed end-to-end analysis of 17 years of highly noisy temporal and spatial water quality data.
* **The Deliverable:** Built statistical models and data visualizations, ensuring project longevity by packaging reproducible code via GitHub and a technical report for nonprofit stakeholders.
